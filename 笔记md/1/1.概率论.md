**贝叶斯公式：**

```
要保证先验与之后有关联性，不是独立的才能用。
比如生男女概率各50%，第一个是男，那么就已经独立与之后无关了。之后概率仍然1/2
如果看整体，得知这个家庭2个人，其中一个是男。那么此时就不独立了，设男A女B，那么总体样本中就没有BB这种情况。AA的情况占1/3，AB的情况占2/3
```

**分布函数与概率密度函数：**

```
定义分布函数为<=x的所有概率之和，因此分布函数单调增
定义概率密度函数，每个点的值代表抽到那个点的概率
对概率密度指定区间积分，就得知那个区间的概率
```

**常见的分布模型：**

```
两点分布，a概率为……，b概率为……，没有其他情况
均匀分布：每个点的概率都相同
二项分布：n次中成功1~n次的各个概率，以及将二项分布逼近下的柏松分布
几何分布：成功就停下的概率
超几何分布：a个合格b个不合格，不放回抽n个，不合格的期望
指数分布：分布函数是e^(-rx)的，其中r是正数。多阶微分方程搞出来的鬼东西
正态分布：也是因为多阶微分方程搞出来的恶心玩意。比如我们可以用@复数来表达出x和y。我们假定x和y共同决定最终的结果。比如g(r)=p(rcos@)*p(rsin@)。同时假设@对最终结果没有产生贡献。比如扔飞镖，用x，y表示扔到的位置，概率是g(r)，假定相同的r，不同的@概率是一致的。这样的想法就导致对@的偏导即d(g(r)/d@为0。也就是导函数处处为0，即各个阶完全独立，函数都形式上相同（可以写成指数e^(-x))之类。
```
